I.Introduction

A. What is CompoDynamics?

The molecular composition and characteristics of coding sequence (CDS) have been shown to affect mRNA stability, translation efficiency, binding capability, structure and function of product. Quantifying molecular compositions for genes is useful for studies in codon optimization, translation, protein structure and function as well as evolution. To integrate such data, we developed CompoDynamics (Database of Molecular Composition Dynamics), a comprehensive database of molecular composition dynamics of genes and genomes. CompoDynamics now represents 36,345 latest reference or complete genomes corresponding to 26,540 organisms, 127,778,183 coding sequences (CDS) of 2,009,532 genes for 75,381,484 proteins collected from RefSeq (see statistics page for more information), and we plan to incorporate more data from GenBank and Ensembl in the future. As for each CDS, we integrate molecular composition features including their nucleotide composition, synonymous codon composition, codon usage, amino acid usage, codon usage bias, coding potential and protein physicochemical properties; For each genome, molecular information including nucleotide composition, codon usage, amino acid usage and codon usage bias is provided in genome scale, which enables cross-species analysis. The detailed molecular composition annotation of CDS/genome and related metadata can be accessed via a user-friendly interface that includes individual report pages for each CDS and genome.

II.Data and methods (link to statistics page)

A.Where does the data come from?

We collect data from RefSeq genomes (ftp site: https://ftp.ncbi.nlm.nih.gov/genomes/refseq/), and download all the coding sequences (CDS) from reference/representative/complete genomes (see RefSeq introduction file for definitions). In addition, only the latest versions are selected.

B.How does CompoDynamics process the data?

We first download “*cds_from_genomic.fna.gz” files from RefSeq, which contain CDS for each genome. Based on these sequences, we then calculate features within 7 classes:

Classes Calculations
nucleotide composition GC, AG, GC1/2/3, AG1/2/3
synonymous codon usage A3s, T3s, C3s, G3s, GC3s
codon usage codon number and frequency
amino acid usage amino acid number and frequency
codon usage bias RSCU, ENC, CDC
coding potential coding potential score, predicted coding label, fickett score
protein physicochemical properties molecular weight, surface area, residue volume, hydrophobicity, charge, structure

Among them, RSCU (Relative Synonymous Codon Usage) values, CDC (Codon Deviation Coefficient) values are calculated by CAT (Zhang et al. 2012); ENC values, synonymous codon compositions are calculated by codonw (John Peden); Fickett scores, coding potential score and predicted coding label are calculated by cpc2 (Kang et al. 2017).

After that, some of the features (nucleotide composition, codon/amino acid usage, codon usage bias) are calculated in genome scale. For instance, GC values are averaged by weights of each CDS length to yield genome GC.

III.How to use CompoDynamics?

A. Browse molecular composition of CDS in CompoDynamics

Users can browse all CDS in the browse page of genes by searching the gene name and protein ID or name of interest. The basic information of CDS’s location, quality, length, GC, AG, GC1/2/3, AG1/2/3 content will be displayed, and the CDS ID is linked to detailed information page of various molecular compositions of CDS.

B. Browse molecular composition of organisms in CompoDynamics

Users can browse all organisms in the browse page of organisms by searching the organism name a of interest. The basic information of organism tax ID, BioProject, BioSample, Refseq category, version status, assembly level and genetic code will be displayed, and the organism ID is linked to detailed information page of various molecular compositions of organism.

C. Perform cross-organisms/species study of molecular compositions in CompoDynamics

In the statistics page, users can compare the distribution of GC, Codon usage bias, codon usage and amino acid usage pattern between different organisms. In addition, users can compare the distribution of coding probability, CDS length, GC/AG content, synonymous codon composition, codon usage bias and protein features within species in the statistics page.
